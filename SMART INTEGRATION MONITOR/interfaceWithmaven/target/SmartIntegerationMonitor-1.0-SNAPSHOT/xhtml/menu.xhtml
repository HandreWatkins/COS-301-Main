  • SMART INTEGRATION MONITOR is a Java EE 7 web application that is used to monitor and observe application servers by collecting and analyzing network related data(such as response time and URI).

    This analysis will provide the user of the system to determine the overall health and performance of the current application server. This will then give the network administrator to correct any errors on the monitored network and improve the performance of the network communications in the application server.
